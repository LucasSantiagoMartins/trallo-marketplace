# TRALLO Marketplace

Plataforma marketplace moderna focada em transformar a experiência de compra e venda em Angola.

O TRALLO foi desenvolvido para oferecer uma experiência inteligente, prática e escalável para clientes e vendedores, criando um ecossistema digital mais eficiente para o comércio online no mercado angolano.

## 📌 Objetivo

Criar uma plataforma marketplace moderna para Angola, focada em melhorar a experiência digital de compra e venda através de tecnologia, usabilidade e escalabilidade.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando:

- ReactJS
- TypeScript
- Vite
- TailwindCSS
- Vitest
- ESLint

---

## 📁 Estrutura do Projeto

Organização base do projeto:

```bash
src/
```
